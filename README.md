# Totem de Autoatendimento — Restaurante Universitário

Interface interativa de um totem de autoatendimento para o Restaurante Universitário do IFSP Campus Pirituba, desenvolvida como projeto prático da disciplina de Interação Humano-Computador.

**[Acessar protótipo](https://sorayagomess.github.io/totem-ru-ifsp/)**

---

## Sobre

O projeto propõe uma solução para os problemas de atendimento presencial no RU — filas longas, falta de visibilidade do cardápio e ausência de informações sobre alergênicos — por meio de uma interface de autoatendimento que permite ao usuário visualizar o cardápio, montar o pedido e realizar o pagamento de forma autônoma.

A interface cobre o fluxo completo: tela inicial, menu de pratos com filtros por categoria, cardápio semanal, carrinho, pagamento (Pix, crédito e débito) e confirmação com número de pedido.

## Tecnologias

HTML · CSS · JavaScript — arquivo único, sem dependências externas.

## Como usar

Abra o arquivo `.html` diretamente no navegador, ou acesse pelo link do GitHub Pages acima.
